# Technology Stack

## Programming Languages

### JavaScript (ES6+)
- **Version**: V8 runtime (modern JavaScript features)
- **Usage**: 
  - Backend logic (Google Apps Script)
  - Frontend interactivity
  - Test scripts
- **Features Used**: Arrow functions, template literals, destructuring, spread operator, async patterns

### HTML5
- **Usage**: User interface markup
- **Features**: Semantic elements, data attributes, form inputs
- **Structure**: Component-based HTML files (.html extension)

### CSS3
- **Usage**: Styling and layout
- **Features**: Flexbox, grid, custom properties, media queries
- **Organization**: Component-specific stylesheets embedded in .html files

## Core Technologies

### Google Apps Script
- **Platform**: Google Workspace automation platform
- **Runtime**: V8 JavaScript engine
- **Version**: V8 runtime (specified in appsscript.json)
- **Services Used**:
  - SpreadsheetApp (Google Sheets integration)
  - HtmlService (web app rendering)
  - ScriptApp (web app deployment)
- **Execution**: Server-side JavaScript on Google infrastructure

### Google Sheets
- **Role**: Database backend
- **Usage**: Data storage, retrieval, and manipulation
- **Access**: SpreadsheetApp API
- **Features**: Multi-sheet support, structured data tables

## Build System and Tools

### Google Clasp
- **Purpose**: Command-line tool for Apps Script development
- **Configuration**: .clasp.json
- **Commands**:
  - `clasp push` - Deploy code to Google Apps Script
  - `clasp pull` - Download code from Apps Script
  - `clasp open` - Open project in Apps Script editor
- **Settings**:
  - Root directory: `src/`
  - Script extensions: `.js`, `.gs`
  - HTML extensions: `.html`
  - JSON extensions: `.json`

### Node.js and NPM
- **Purpose**: Package management
- **Configuration**: package.json
- **Dependencies**:
  - `@modelcontextprotocol/server-github` (v2025.4.8)

### Git
- **Purpose**: Version control
- **Configuration**: .gitignore
- **Workflow**: Standard Git workflow with GitHub integration

## Frontend Technologies

### Vanilla JavaScript
- **No Framework**: Pure JavaScript without React/Vue/Angular
- **DOM Manipulation**: Native DOM APIs
- **Event Handling**: addEventListener patterns
- **AJAX**: google.script.run for server communication

### CSS Architecture
- **Methodology**: Component-scoped styles
- **Layout**: Flexbox and CSS Grid
- **Responsive**: Media queries for mobile support
- **RTL Support**: Right-to-left text direction for Arabic

### UI Components
- **Font Awesome**: Icon library (referenced via CDN)
- **Custom Components**: FormModal, ViewTab
- **No UI Framework**: Custom-built components

## Development Commands

```bash

# Pull code from Google Apps Script

# Open project in browser
clasp open
```

### Version Control
```bash

```

### Testing
- Manual testing via Dashboard.test.html

- Browser-based UI testing

## API and Integration

### Google Apps Script API
- **google.script.run**: Client-to-server communication
- **Success/Failure Handlers**: Callback-based async patterns
- **Parameter Passing**: JSON serialization for complex objects

### Web App Deployment
- **URL**: Generated by Google Apps Script
- **Protocol**: HTTPS
- **Authentication**: Anonymous access enabled
- **Execution**: Runs as deploying user

## Browser Compatibility
- **Target**: Modern browsers (Chrome, Firefox, Safari, Edge)
- **JavaScript**: ES6+ features
- **CSS**: Modern CSS3 features
- **No IE Support**: Assumes modern browser environment
